# Calculadora em Python (Tkinter)

Este projeto é uma calculadora simples com interface gráfica desenvolvida em Python utilizando a biblioteca Tkinter.

## Funcionalidades
- Operações básicas: soma, subtração, multiplicação e divisão
- Suporte a parênteses
- Interface moderna inspirada em calculadoras mobile
- Botão de limpar (C)
- Tratamento de erros

## Tecnologias utilizadas
- Python
- Tkinter (interface gráfica)

## Como executar
1. Instale o Python
2. Execute o arquivo:
   python nome_do_arquivo.py

## Objetivo
Projeto desenvolvido para praticar lógica de programação e criação de interfaces gráficas em Python.
